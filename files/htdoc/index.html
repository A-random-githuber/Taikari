Taikari X-Random-Challenge API

ATTENTION! This API is for everyone, Please DO NOT abuse this service.

GET /v1/generate

arguments description optional
method GET or POST NO
path request url including the domain NO
body post body YES, while the method is GET

Explain

Generates an X-Random-Challenge string.

Return value

{
    "status": 0,
    "content": {
        challenge: "xxxx"
    }
}

Error codes

status description
0 everything is OK
